# 聊天记录合并转发插件 for AstrBot

## 功能特点

- ✅ 支持将当前聊天的指定时间范围消息转发到QQ好友/QQ群
- ✅ 支持从指定QQ好友/QQ群转发消息到当前聊天
- ✅ 支持转发到多个目标（使用 | 分隔）
- ✅ 灵活的时间范围选择（相对时间/绝对时间）
- ✅ 完善的管理员权限系统
- ✅ 支持合并转发（QQ内置的聊天记录样式）
