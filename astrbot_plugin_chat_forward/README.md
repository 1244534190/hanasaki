# 聊天记录合并转发插件 for AstrBot

## 功能特点

- ✅ 支持将当前聊天的指定时间范围消息转发到QQ好友/QQ群
- ✅ 支持从指定QQ好友/QQ群转发消息到当前聊天
- ✅ 支持转发到多个目标（使用 | 分隔）
- ✅ 灵活的时间范围选择（相对时间/绝对时间）
- ✅ 完善的管理员权限系统
- ✅ 支持合并转发（QQ内置的聊天记录样式）

## 安装方法

克隆仓库到AstrBot的插件目录：
```bash
cd astrbot/plugins
git clone https://github.com/hanasaki/astrbot_plugin_chat_forward.git

安装依赖：

pip install -r requirements.txt

重启AstrBot

## 配置说明

配置文件位于 data/plugins/chat_forward/config.json

json
{
  "admins": ["管理员QQ号"],
  "super_users": ["超级管理员QQ号"],
  "max_forward_count": 100,
  "enable_emoji": true,
  "forward_delay": 0.5
}

# 使用命令

## 转发到目标（转去）
将当前聊天的历史消息转发给其他好友或群：

/转去 目标 时间范围

# 示例：

/转去 123456 30分钟前 - 转发最近30分钟的消息给好友123456

/转去 g789456 2小时前 - 转发最近2小时的消息给群789456

/转去 123456|g789456 2024-01-01 到 2024-01-31 - 转发指定时间范围的消息给多个目标

# 从源转发（转来）
从其他好友或群获取消息转发到当前聊天：

/转来 源 时间范围
示例：

/转来 123456 30分钟前 - 从好友123456获取最近30分钟的消息

/转来 g789456 2小时前 - 从群789456获取最近2小时的消息

# 管理员管理

/添加管理员 QQ号        # 添加管理员（需要超级管理员）
/移除管理员 QQ号        # 移除管理员（需要超级管理员）
/管理员列表            # 查看管理员列表
时间格式支持
相对时间
30分钟前

2小时前

3天前

1周前

1月前

绝对时间
2024-01-01 到 2024-01-31

2024-01-01 10:00 到 2024-01-01 18:00

# 注意事项
1.需要确保AstrBot有消息历史记录功能
2.转发大量消息可能受QQ风控影响
3.合并转发功能需要QQ客户端支持
4.超级管理员默认拥有所有权限
